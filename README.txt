Klickon Auth Demo
=================
Bu klasör, basit bir Flask tabanlı Login/Register uygulaması içerir.

Çalıştırma:
1) Sanal ortam oluşturun ve aktif edin.
2) pip install -r requirements.txt
3) python app.py
4) http://127.0.0.1:5000 adresini açın.

Not: Şifreler bcrypt ile saklanır. Session/token kullanılmamaktadır (görev gereği).
